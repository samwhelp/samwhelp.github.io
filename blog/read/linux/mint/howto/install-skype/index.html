Linux Mint 17 “Qiana” Cinnamon 64位元 的環境，如何安裝「Skype」。

測試環境

  • 「Linux Mint 17 “Qiana” Cinnamon 64位元」
  • 「Cinnamon 2.2.16」

緣起

在「上一篇已經解決麥克風的問題了」，再來就是要解決「在64位元安裝Skype」的問題。基本上這兩個問題，在以前我安裝「Xubuntu 14.04 64位元」那時候都遇到過，也解決過，不過哪時候都沒有紀錄下來，只是隱約有印象怎麼解決的。另外這裡的作法，也同樣可以解決安裝「Teamviewr」的問題。也可以解決「泛Ubuntu 14.04 64位元的版本」安裝這兩個軟體，所會遇到的問題。就不會遇到「sudo apt-get install -f」為了安裝相依的套件(32位元)，而把64位元的套件給移除了。

步驟

加入支援32位元。

這個步驟在「Linux Mint 17 “Qiana” Cinnamon 64位元」，應該是不用執行，預設好像就有了。

安裝 Deb檔
