Linux Mint 17 “Qiana” Cinnamon 的環境，如何讓 microphone (麥克風) 願意發聲。

測試環境

  • 「Linux Mint 17 “Qiana” Cinnamon」
  • 「Cinnamon 2.2.16」

緣起

發現我的麥克風沒有作用，在「System Settings / Hardware / Sound / Input」也無法設定。

用「linux mint microphone not working」，找到兩篇文章「1」，「2」。

主要是第二篇)的說明。

步驟

安裝「gnome-media」

文章中提到要執行「gstreamer-properties」這個指令，不過在Linux Mint 17 “Qiana” Cinnamon 的環境，預設並沒有安裝，查了一下是屬於「gnome-media」這個套件。

安裝完後，就可以執行「gstreamer-properties」這個指令。

然後就照文章的指引，
在「Audio / Default Input / Plugin: 」選「ALSA - Advanced Linux Sound Architecture」。
在「Device:」選擇你「麥克風裝置」的「型號」就行了。
可以按下「Test」那個按鈕來測試，是否有效。

安裝「gnome-alsamixer」

另外在系統原本的聲音設定「System Settings / Hardware / Sound / Input」還是沒有出現。

這時候就要在安裝另一個套件「gnome-alsamixer」。

安裝完畢後可以執行「gnome-alsamixer」，
或是在程式選單也可以找到，用關鍵字「Alas」查，比較快找到摟，就會看到「GNOME ALSA Mixer」這個選項。
未來應該會常用到，為了方便，也可以在這個選項按下右鍵「Add to Panel」或「Add to favorites」。

然後找到「Front Mi」或「Rear Mic」，取消「Mute」的勾選。調整聲量大小。
microphone (麥克風) 就會願意發聲了。
我的「耳麥」是裝在前面，所以要取消「Front Mi」的「Mute」。
