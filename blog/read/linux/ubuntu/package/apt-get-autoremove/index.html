關於「apt-get autoremove」

關於「apt-get autoremove」

測試環境

以下測試是在「Xubuntu 14.04 64位元」。

前提

有時候執行安裝套件的時候

就會出現類似下面的訊息

解決方式

上面的提示訊息已經有說明解法了。

就會把一些上面列的套件移除，並且不會在出現上面的提示訊息。

說明

會出現這樣的提示訊息，就是有些套件的安裝，是伴隨在安裝某些套件，依據相依性而安裝的。
後來移除了，這些依據相依性而安裝的套件，如果沒有被其他的套件相依，就會出現這樣的提示訊息。
