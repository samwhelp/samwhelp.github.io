是透過「dpkg」這個指令，來安裝「Debian Package」。

以安裝「hello」這個套件為例。

先下載「hello_2.8-4_amd64.deb」這個「Debian Package」。

利用「dpkg」來安裝「hello_2.8-4_amd64.deb」
