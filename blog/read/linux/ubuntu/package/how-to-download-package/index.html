注意：測試的結果

  • 當hello沒有安裝的時候，加上「-d」才會下載，若已經安裝hello，加上「-d」則不會執行下載。
  • 當相依的套件若尚未安裝，也會一併下載。
  • 然後加上「–reinstall」，雖然已經安裝，但還是會下載。
