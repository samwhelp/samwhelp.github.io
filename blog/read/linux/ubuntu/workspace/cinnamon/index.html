如何新增和切換 Cinnamon 的 Workspace

如何新增和切換 Cinnamon 的 Workspace

前言

昨天開始認真的試玩「Cinnamon」，這個桌面環境。
我是安裝「Linux Mint 17 “Qiana” Cinnamon」這個版本。

想要使用「Workspace」這個功能。
在「Panel」按下右鍵，出現選單，選「Add applets to the Panel」，出現「Themes」這個設定視窗。
在「查詢(Filter)」那個控制項，輸入「workspace」，就會看到一筆「Workspace switcher」。點兩下，就會出現在「Panel」。
一開始預設只有兩個「Workspace」可以切換。按下右鍵，出現的選單，也沒有可以設定。只有一個選項可選「Remove this applet」。
這時候只有到「System Settings」去找摟。
在「System Settings / Preferences / Workspaces」，沒有看到，可以新增Workspace的選項。

這時候我就聯想到「Cairo-Dock」，我每次都會把「Switcher」這個「Add-ones」給停掉。
這時候，就把他給重新打開。結果在「Switcher」按下右鍵。有「Add a workspace」這個選項，點下執行，真的新增了耶。呵呵。

後來想說，去找看看有沒有相關的文件，紀錄是否有「shortcut key」可以新增或切換「Workspace」。
目前我在「Linux Mint」和「Cinnamon」找，還沒找到。

另外用「workspace cinnamon」，倒是有找到一篇有用的「討論」。

Shortcuts

就是按下「Ctrl+Alt+Up」，就會出現一個全螢幕的畫面，裡面可以列出所有的「Workspace」。這篇文章，就有類似的圖片。
右方還有一個諾大「+」號的按鈕，可以新增Workspace。
滑鼠游標移到某個「Workspace」，在右上方，還會出現「x」的圓形按鈕，讓你可以關閉這個「Workspace」。
也可以用鍵盤按「上下左右」來移動你關注的「Workspace」。按下「Delete」也可以刪除該「Workspace」。

這篇文章也有詳細的說明。
還有一個「「Ctrl+Atl+Down」的快捷健。
和自訂workspce的title。

然後在一般的桌面畫面。可以使用「Ctrl+Atl+Left」，「Ctrl+Atl+Right」，可以切換不同的「Workspace」。
我特地是嘗試不同的桌面系統，「Ctrl+Atl+Left」，「Ctrl+Atl+Right」這兩個好像是通用的快速鍵，用來切換「Workspace」。「kde」目前測試應該是要自己設定，也很好設定。

Key binding

在「Cinnamon」，可以在「System Settings / Hardware / Keyboard / Keyborad shortcuts / Workspace management 」設定。

小結
