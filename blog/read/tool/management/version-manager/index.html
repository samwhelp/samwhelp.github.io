關鍵詞彙

###「vm」

  • 「nvm」 - 「nodejs」
  • 「rvm」- 「ruby」

「env」

都是從「rbenv」衍生出來的。

  • 「rbenv」-「ruby」
  • 「pyenv」-「python」
  • 「plenv」-「perl」
  • 「phpenv」-「php」

「brew」

  • 「pythonbrew」-「python」

  • 「perlbrew」-「perl」

  • 「phpbrew」-「php」

###「virtual」

  • 「virtualenv」-「python」
  • 「virtphp」-「php」
