Brackets Howto

Brackets Howto

測試環境

以下測試環境在
「Xubuntu 14.04 64位元」 「Brackets.Release.0.44.64-bit」

Ubuntu底下的安裝方式，請參考「這篇」。

如何切換界面語系

功能選單「Debug / Switch Language」，會跳出一個「Switch Language」對話框。有一個「Language:」的下拉選單，可以選。
選好你要的語言，例如「繁體中文」。按下「Reload Brackets」右下角這個按鈕。界面的文字就會改變了。
預設的選項是「System Default(系統預設)」。

如何開啟設定檔

功能選單「Debug / Open Preferences File」。
會開啟設定檔，這個設定檔是「~/.config/Brackets/brackets.json」

如何開啟「Extension Manager(擴充功能管理員)」

  • 功能選單「File / Extension Manager…」
  • 編輯區的右方，功能bar。在「閃電」圖示下，有一個圖示，按下它，一樣可以。

詳細參考「這裡」。

如何透過「Extension Manager」來安裝「Extension」.

從上面的方式，呼叫起「Extension Manager」對話框。會分成三個頁簽，「Available」，「Themes」，「Installed」，右方還有一個「搜尋控制項」。
找尋你要安裝的「Extension」，按下右方的「Install」按鈕，就會安裝該「Extension」.

如何找到安裝好的「Extension」放在哪個資料夾。

功能選單「Help / Show Extensions Folder」。就會開啟系統的檔案總管，指到那個路徑「~/.config/Brackets/extensions/」。
透過上面的方式，裝好的「Extension」則是放在「~/.config/Brackets/extensions/user」。

如何選擇不同的「Theme」。

功能選單「View / Themes…」。
開啟「Themes Settings」對話框。
有一個「Current Theme:」下拉選單。
下拉選擇你要的「Theme」，就會即時反應。
按下右下角的「Done」按鈕，關閉「Themes Settings」對話框。
