前言

今天從「nodeschool」看到了「Brackets」這個對我來說，比較新的編輯器。

原本興趣還沒有那麼濃，直到知道它和「Atom」一樣，也是「Nodejs」平台打造架設起來的。並且第一個裝的「Plugin」-「Markdown Preview」，引起了我的興趣，就開始玩起勁來了。本來還嘗試把我目前「Hexo」使用的佈景的主題的CSS，帶入Plugin裡，結果尚未成功。

比較詳細的介紹，請參考「這篇」或是「這篇」或是查詢「brackets」，找到相關的文章。

會出現

主要是安裝在「/opt/brackets」這個資料夾。

「/usr/bin/brackets」是一個「symbolic link」到「/opt/brackets/brackets」

預設的「extension」放在「/opt/brackets/www/extensions/default/」

整個編輯器是「/opt/brackets/www/index.htm」這個頁面。
